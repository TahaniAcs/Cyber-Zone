# Cyber-Zone
🛡️ Cyber Zone - Educational Game

Cyber Zone is an interactive web-based platform designed to teach essential cybersecurity concepts through engaging gameplay and real-world simulations.  
🎮 Game Challenges

The project features five distinct challenges, each focusing on a specific security domain:

    Challenge 1: Cyber Escape (Phishing) - Learn to identify malicious links and suspicious email patterns.  

    Challenge 2: Cyber Cross (Crossword) - Test your knowledge of cybersecurity terminology in a fun grid format.  

    Challenge 3: Social Engineering Chat - Practice blocking manipulative prompts and protecting sensitive info.  

    Challenge 4: Word Order Quest - Reassemble scrambled letters to reveal key security concepts and earn badges.  

    Challenge 5: Spot the Risk - Scan an office environment to identify physical security threats and unsafe behaviors.  

🛠️ Technical Stack

    HTML5 & CSS3: Responsive UI inspired by modern corporate design standards (SABIC-style theme).  

    Vanilla JavaScript: Core logic for interactive chat, drag-and-drop assessments, and score tracking.  

📂 Repository Structure

To ensure the game functions correctly, please maintain the following file structure in the root directory:

    index.html: Main gateway and module selector.  

    2.html - 6.html: Individual challenge pages.  

    go.html: Mission completion and achievement screen.  

    1.css: Global stylesheet for consistent branding.  

    /assets: Directory for images (e.g., LOG2.png, 3.png).
